# K40 Laser



Utilisation de l'eau déminéralisée

Chasser les bulles d'air du tube C02

Format A4

Logiciel Inkscape

**3 types de gravure**

* Raster (image) - Noir
* Découpe - Rouge
* Vecteur - Bleu
